# AI Chatbot - .NET Windows Forms

A Windows Forms AI Chatbot powered by Claude API.

## Technologies
- .NET 10.0
- Windows Forms
- Claude API (Anthropic)

## Setup Instructions
1. Clone the repository
2. Open `AIChatbot.sln` in Visual Studio
3. Replace `YOUR_CLAUDE_API_KEY_HERE` in `ApiService.cs`
4. Press F5 to run

## Features
- Real-time AI chat
- Clean UI
- Error handling
- Enter key support
